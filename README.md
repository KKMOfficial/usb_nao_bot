# usb_nao_bot
feature development for nao bot using reinforcement techniques and more...


One of the primary aim's of this project is to develope a continous time controller to control the walking of the humanoid bot. We choose nao as the representative model in the webots environment. in the first trial, we are going to train actor-critic network for the robot using DDPG method.

Read the DDPG paper from [Here](https://arxiv.org/abs/1509.02971)

To gain more information about model refer to [SoftBank Robotics' Nao](https://cyberbotics.com/doc/guide/nao)
